# Página web de Muebles Minimalistas

Este es el repositorio para mi proyecto final del curso de Desarrollo Web de Coderhouse. He creado una página web para una tienda de muebles minimalistas utilizando HTML, CSS, SASS, Bootstrap, técnicas de SEO, Git y BEM.

## Estructura del Sitio

- **Página de inicio**: La página principal del sitio, que incluye enlaces internos a diferentes secciones.
- **Navbar**: Una barra de navegación que se encuentra en la parte superior de cada página y permite al usuario acceder fácilmente a diferentes secciones del sitio.
- **Pestaña de Productos**: Una sección que muestra una selección de productos utilizando tarjetas (cards) con efectos de transición.
- **Pestaña de Promociones**: Una sección que muestra promociones especiales utilizando un carrusel (carousel) de Bootstrap.
- **Acerca de Nosotros (About Us)**: Una sección que proporciona información sobre la empresa y su filosofía. Incluye enlaces a Google Maps para mostrar la ubicación física y a YouTube para ver videos relacionados.
- **Formulario de Contacto**: Una sección que permite a los usuarios enviar consultas o pedidos a través de un formulario de contacto.

## Tecnologías utilizadas

- HTML: El lenguaje de marcado utilizado para estructurar el contenido de la página.
- CSS y SASS: Utilizados para dar estilo y diseño a la página.
- Bootstrap: Framework de CSS utilizado para agilizar el desarrollo y lograr un diseño responsivo.
- Técnicas de SEO: Implementadas para mejorar la visibilidad de la página en los motores de búsqueda.
- Git: Sistema de control de versiones utilizado para el seguimiento de cambios y colaboración.
- BEM (Block, Element, Modifier): Metodología de nomenclatura utilizada para escribir código CSS más modular y mantenible.

## Instrucciones de instalación

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web.

¡Y eso es todo! Ahora podrás explorar mi página web de muebles minimalistas y ver todas las estructuras que he creado.

Si tienes alguna pregunta o sugerencia, no dudes en contactarme. ¡Espero que disfrutes de la experiencia!

# Minimalist Furniture Website

This repository contains the final project for the Coderhouse Web Development course. I have created a website for a minimalist furniture store using HTML, CSS, SASS, Bootstrap, SEO techniques, Git, and BEM.

## Site Structure

- **Home Page**: The main page of the website, featuring internal links to different sections.
- **Navbar**: A navigation bar located at the top of each page, allowing users to easily access different sections of the site.
- **Products Tab**: A section that showcases a selection of minimalist furniture items using transition effects on cards.
- **Promotions Tab**: A section that displays special promotions using a Bootstrap carousel.
- **About Us**: A section that provides information about the company and its philosophy. It includes links to Google Maps for displaying the physical location and to YouTube for watching related videos.
- **Contact Form**: A section that allows users to submit inquiries or place orders through a contact form.

## Technologies Used

- HTML: Markup language used to structure the page content.
- CSS and SASS: Used for styling and designing the website.
- Bootstrap: CSS framework used for a responsive design and rapid development.
- SEO Techniques: Implemented to enhance the website's visibility on search engines.
- Git: Version control system used for change tracking and collaboration.
- BEM (Block, Element, Modifier): Naming methodology used for writing modular and maintainable CSS code.

## Installation Instructions

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.

That's it! Now you can explore the Minimalist Furniture Website and experience the different sections and features I have implemented.

If you have any questions or suggestions, feel free to contact me. I hope you enjoy browsing the website!
